# Arduino Memory Game

A Simon-like memory game made usng the Arduino. 

## Materials

- Breadboard
- Arduino Uno
- Around 20 - 25 jumper wires
- 8 220ohm resistors
- 4 pushbuttons
- 4 LEDs
- One LCD with I2C module
